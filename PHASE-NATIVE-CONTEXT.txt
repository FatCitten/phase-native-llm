SECTION 1: THE CORE THESIS
================================================================================

Contemporary machine learning systems are MAP-LIKE. They store POSITIONS
(coordinates of concepts in learned representation space) and generalize by
INTERPOLATING between known positions.

Aaron's thesis: This is the wrong primitive.

The correct primitive is not POSITION but CONNECTION — the geometric rule
governing how representations TRANSFORM as context shifts. This is the
difference between storing WHERE you are on a curved surface vs storing
the CURVATURE RULES of the surface itself.

One sentence summary:
"Current neural networks are maps. Phase-native systems are instruments
for reading the geometry of meaning."

SECTION 12: GLOSSARY (for Claude Code sessions)
================================================================================

HOLONOMY:
The rotation/transformation a vector accumulates when parallel-transported
around a closed loop on a curved surface. Encodes total enclosed curvature.
In NN terms: the transformation error around a cycle in the computation graph.

PARALLEL TRANSPORT:
Moving a vector along a curved surface while keeping it "as parallel as
possible" to itself. The path matters — different paths give different results
on curved surfaces. This is what attention should approximate.

FIBER BUNDLE:
A space E that locally looks like B × F (base × fiber) but may be globally
twisted. The connection ∇ defines how fibers relate across base points.
In NN terms: contexts are base points, representations are fibers,
attention/weights approximate the connection.

CONNECTION ∇:
The geometric rule governing parallel transport between fibers.
NOT a position. A TRANSFORMATION RULE.
This is what phase-native systems learn instead of coordinates.

PHASE:
The complex argument of a complex-valued weight or activation.
Encodes rotational/directional information that scalar weights discard.
When you go from real to complex weights you recover phase information.

CONTEXTUALITY INDEX κ:
From arxiv.org/pdf/2509.10536. Scalar measure of how non-trivial the
holonomies are in a network. κ = 0 means flat (current ML). κ > 0 means
geometric structure is present. Phase-native training should maximize
STRUCTURED κ (coherent curvature, not random curvature).

RIEMANNIAN GRADIENT DESCENT:
Gradient descent performed on a curved manifold rather than flat Euclidean
space. Gradients must be projected onto the tangent space of the manifold
at each step. Implemented in Geoopt library.

BERRY PHASE:
A phase acquired by a quantum system when its parameters are transported
around a closed loop in parameter space. The original physical holonomy.
The mathematical object that inspired this entire framework.

- Output includes both magnitude AND phase

STEP 2: HolonomyLoss function
──────────────────────────────
Compute holonomy error per layer:
- For each layer, define a canonical cycle (e.g., input → hidden → output → input)
- Compute the composition of transformations around the cycle
- Measure deviation from identity (or from expected curvature)
- This becomes the geometric term in the loss

STEP 3: PhaseCoherentTransformer
──────────────────────────────────
Stack PhaseCoherentAttention blocks
Add standard feed-forward layers (keep these real-valued initially)
Add HolonomyLoss computation at each layer
Total loss = Σ holonomy_errors + λ * cross_entropy
